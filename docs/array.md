# Go 语言数组实现原理知识图谱

根据《Go语言设计与实现》整理，涵盖了数组的类型定义、编译优化、内存布局及访问机制。

## 核心知识点摘要

1.  **类型定义**：由**元素类型**和**大小**共同决定。`[3]int` 和 `[4]int` 在 Go 中被视为不同类型。
2.  **初始化优化**：
    *   **≤ 4 个元素**：直接在**栈**上初始化，拆分为独立赋值语句。
    *   **> 4 个元素**：在**静态存储区**初始化临时变量，再整体拷贝到栈。
3.  **越界检查**：
    *   **编译时**：检查常量索引（非整数、负数、越界）。
    *   **运行时**：检查变量索引，插入 `IsInBounds` 和 `PanicBounds` 指令。
4.  **底层操作**：编译器将访问/赋值转换为内存地址计算（`LocalAddr`、`PtrIndex`）和直接读写（`Load`、`Store`）。
